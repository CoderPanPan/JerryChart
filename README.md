# JerryChart
Android chart 饼状图简单实现
扇形、环形饼状图，显示百分比文本在扇形、环形上，实现扇形分离，扇形、环形分割线
代码较为简单粗糙，百分比只显示整数百分比小数位舍去，有需要的请自行修改
